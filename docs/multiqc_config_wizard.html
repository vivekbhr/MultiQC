    Welcome to the MultiQC Config Wizard

    This tool helps you create a custom multiqc_config.yaml configuration file. You don't need to set every option, only change what you want to customise. Unset options will use their defaults.

    Browse sections in the sidebar, or use the search bar above to find a specific option. When you are done, click View YAML to see the generated config. You can also paste an existing multiqc_config.yaml there to have each option validated and explained.
